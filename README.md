# Wenn dein Dokument einen Ausweis braucht

Interaktive QR-Vertiefung für Band 3 / Kapitel 2.

## Dateien

- `index.html`
- `styles.css`
- `script.js`
- `README.md`

## Veröffentlichung

1. Neues GitHub-Repository erstellen oder bestehendes QR-Repository nutzen.
2. Dateien in den Hauptordner hochladen.
3. GitHub Pages aktivieren: Settings → Pages → Deploy from a branch → main → /root.
4. Die `github.io`-Adresse als QR-Ziel verwenden.
